This directory contains the python program to generate Figs 2B, 2C, 2D and 2E in "Hidden structure in disordered proteins is adaptive to intracellular changes" (Moses et al, 2021, www.biorxiv.org/content/10.1101/2021.11.24.469609v1), as well as all dependent files. Copy this directory to your computer and run the python script, and it will generate Figs 2B, 2C, 2D and 2E. Contact David Moses at dmoses5@ucmerced.edu with any questions.
