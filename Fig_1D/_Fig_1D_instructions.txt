This directory contains the python program and all necessary files to generate Fig 1D in "Hidden structure in disordered proteins is adaptive to intracellular changes" (Moses et al, 2021, www.biorxiv.org/content/10.1101/2021.11.24.469609v1). Copy this directory to your computer and run the python script, and it will generate Fig 1D. Contact David Moses at dmoses5@ucmerced.edu with any questions.
